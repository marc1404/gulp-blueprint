## Installation
```
$ npm install --save-dev gulp-blueprint
```
  
## Usage
*gulpfile.js*
```javascript
var blueprint = require('gulp-blueprint');

blueprint();
```
  
## Test
```
$ npm install -g mocha  
$ mocha
```
